=== Description ===

Purge Cloudflare Cache of the Post with additional URLs on the Post update, Purge Everything button.

Plugin requires Cloudflare account.

Feel free to create PR to [plugin Github repo](https://github.com/gdidentity/c-purge-cache).

== Installation ==

1. Search for the plugin in WordPress under "Plugins -> Add New".
2. Click the “Install Now” button, followed by "Activate".
3. Fill out Cloudflare credentials in Settings -> Cloudflare Purge Cache
